# AutoLogin

Программа для автоматического входа на образовательный портал.

## Особенности
- Автоматический вход на сайт https://poo.edu-74.ru/
- Сохранение учетных данных
- Кроссплатформенность (разработка на Linux, запуск на Windows)
- Автоматическая сборка EXE через GitHub Actions

## Использование
1. Запустите `AutoLogin.exe`
2. Введите логин и пароль при первом запуске
3. Программа откроет Chrome и выполнит автоматический вход

## Для разработчиков
- Код написан на Python 3.8+
- Зависимости: Selenium, webdriver-manager
- Сборка EXE: PyInstaller
